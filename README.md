# Smart Automated No Dues System

An AI-powered No Dues Management System developed using Flask, SQLite, OCR, and Web Technologies to automate student clearance and faculty verification processes.

## Features

- Student Login & Dashboard
- Faculty Login & Dashboard
- Assignment Upload System
- OCR-Based Assignment Verification
- Attendance Clearance Verification
- Department Clearance Management
- No Dues Certificate Generation
- Faculty-wise Approval System
- Secure Authentication
- Responsive User Interface

## Installation

1. Clone Repository

```bash
git clone https://github.com/Manavcar07/Smart-Automated-No-Dues-System.git
```

2. Install Dependencies

```bash
pip install -r requirements.txt
```

3. Run Project

```bash
python app.py
```

4. Open Browser

```bash
http://127.0.0.1:5000
```
